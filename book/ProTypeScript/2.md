# 2章 基本的な文法・基本的な型

# 2.1 文と式

```js
// 文(sentence)
// TypeScript の文は基本的に ;(セミコロン)で終わる
const greeting = "Hello, ";
const target = "world!";
console.log(greeting + target);

// 式(expression)
"Hello, "
greeting + target

// 式文(式のあとにセミコロンを書く文)
console.log(greeting + target);
```

# 2.2 変数の宣言と使用

```js
// const: 再代入不可
// 極力 const を使って変数を宣言すべき

// 型注釈なし 
const greeting = "Hello, ";

// 型注釈あり
const greeting: string = "Hello, ";
```

```js
// let: 
// 再代入可
let greeting = "Hello, "; 
greeting = greeting + "world!";

// 宣言時に値を代入しなくてもよい
let greeting, target;
greeting = "Hello, ";

// 型注釈あり
let greeting: string, target: string;
```
